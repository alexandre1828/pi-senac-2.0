📋 Módulo 1: Organização de Documentos

0% Concluído
1

Identificação dos Tipos de Documentos

Objetivo: Aprender a identificar e classificar diferentes tipos de documentos administrativos.

🗂️ Principais Tipos de Documentos:

  • Contratos: Acordos formais entre partes
  • Faturas: Documentos de cobrança
  • Relatórios: Documentos informativos
  • Correspondências: E-mails, cartas, ofícios
  • Comprovantes: Recibos, notas fiscais

💡 Exemplo Prático:

Você recebeu uma pilha de documentos. Vamos classificá-los:

📄
Nota Fiscal nº 12345

Tipo: Comprovante Fiscal

📋
Contrato de Prestação de Serviços

Tipo: Contrato

2

Sistema de Classificação

Objetivo: Criar um sistema eficiente de classificação por categorias.

🏷️ Sistema de Cores e Códigos:

Vermelho - Urgente

Documentos que precisam de ação imediata

Laranja - Financeiro

Faturas, recibos, comprovantes

Verde - Contratos

Acordos e documentos legais

Azul - Correspondências

E-mails, cartas, comunicados

💡 Dica Profissional:

Use etiquetas coloridas ou pastas com cores para facilitar a identificação visual rápida!

3

Organização Física e Digital

Objetivo: Implementar um sistema de arquivamento eficiente.

📁 Arquivamento Físico

  1. Separe por categoria usando o sistema de cores
  2. Organize em ordem cronológica (mais recente na frente)
  3. Use divisórias com etiquetas claras
  4. Mantenha uma lista de controle

💻 Arquivamento Digital

  1. Crie pastas com nomes padronizados
  2. Use formato: "AAAA-MM-DD_TipoDoc_Descrição"
  3. Faça backup regular dos arquivos
  4. Mantenha uma planilha de controle

📞 Módulo 2: Atendimento Telefônico Profissional

0% Concluído
1

Preparação para o Atendimento

Objetivo: Preparar-se adequadamente antes de atender chamadas telefônicas.

🎯 Antes de Atender:

📝
Tenha sempre à mão:

Papel, caneta, agenda, lista de ramais

💻
Sistema atualizado:

Acesso ao sistema da empresa funcionando

🎧
Ambiente adequado:

Local silencioso, sem ruídos de fundo

🗣️ Preparação da Voz:

  • Fale de forma clara e pausada
  • Mantenha um tom amigável e profissional
  • Sorria ao falar (isso transparece na voz!)
  • Mantenha boa postura
2

Script de Atendimento

Objetivo: Dominar as frases e abordagens corretas para cada situação.

📞 Atendimento Inicial

"Bom dia/Boa tarde, [Nome da Empresa], meu nome é [Seu Nome]. Como posso ajudá-lo?"

✅ Sempre se identifique
✅ Seja cordial e direto
✅ Demonstre disponibilidade

🔄 Transferindo Chamadas

"Vou transferir sua ligação para o setor responsável. Por favor, aguarde um momento."

✅ Explique o motivo da transferência
✅ Peça para aguardar
✅ Confirme se a pessoa pode esperar

📝 Anotando Recados

"Posso anotar um recado? Preciso do seu nome completo, empresa e telefone para contato."

✅ Confirme os dados
✅ Repita as informações
✅ Garanta que entregará o recado

3

Situações Desafiadoras

Objetivo: Saber como lidar com situações difíceis no atendimento.

😤 Cliente Irritado

Como agir:

  1. Mantenha a calma e seja empático
  2. Ouça sem interromper
  3. Peça desculpas pelo inconveniente
  4. Ofereça soluções ou transfira para supervisor
Frase exemplo: "Entendo sua frustração e peço desculpas pelo inconveniente. Vou fazer o possível para resolver isso."

❓ Não Sei a Resposta

Como agir:

  1. Seja honesto sobre não saber
  2. Ofereça-se para descobrir
  3. Dê um prazo para retorno
  4. Anote os dados para contato
Frase exemplo: "Não tenho essa informação no momento, mas vou verificar e retorno em até 2 horas. Posso anotar seu telefone?"

📞 Linha com Ruído

Como agir:

  1. Informe sobre o problema na linha
  2. Peça para repetir informações importantes
  3. Ofereça para retornar a ligação
  4. Confirme dados por e-mail se necessário
Frase exemplo: "Estou com dificuldade para ouvir devido ao ruído na linha. Posso retornar sua ligação?"

1

Estrutura do E-mail Profissional

Objetivo: Aprender a estrutura correta de um e-mail profissional.

📝 Assunto

Deve ser claro, específico e informativo

✅ Bom: "Reunião de projeto - 15/03 às 14h"

❌ Ruim: "Reunião"

👋 Saudação

Formal mas cordial

Formal: "Prezado(a) Sr.(a) Silva"

Semiformal: "Olá, João"

Equipe: "Bom dia, pessoal"

📄 Corpo do E-mail

1º Parágrafo: Contexto/Motivo
2º Parágrafo: Informações principais
3º Parágrafo: Ação esperada/Próximos passos

👋 Despedida

Profissional e cordial

"Atenciosamente,"

"Cordialmente,"

"Abraços," (mais informal)

2

Boas Práticas e Etiqueta

Objetivo: Dominar as regras de etiqueta para comunicação profissional por e-mail.

✅ O que FAZER

  • Responder em até 24 horas
  • Usar "Responder a todos" apenas quando necessário
  • Revisar antes de enviar
  • Usar formatação simples e clara
  • Incluir assinatura profissional
  • Confirmar recebimento de e-mails importantes
  • Usar "Com cópia" e "Com cópia oculta" adequadamente

❌ O que NÃO FAZER

  • Usar CAPS LOCK (parece que está gritando)
  • Enviar e-mails com erros de português
  • Usar linguagem muito informal
  • Enviar anexos muito pesados
  • Esquecer de incluir anexos mencionados
  • Usar emoticons em excesso
  • Marcar como "Urgente" sem necessidade
